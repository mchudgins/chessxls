Lightweight Workflow & Content Management using Google Infrastructure

This web application demonstrates the concept of leveraging Google Docs and Google App Engine to provide a light-weight workflow and content management application. The application runs on Google's hosting service. All workflow related content, documents, images, and files, are stored in the Google Docs repository.
